Code Example Gallery

Browse and run every MediaPipe demo in the project — from basic hand tracking to a combined hands and face mesh overlay. Each demo opens directly in your browser and uses your webcam in real time.

⚠ Webcam access requires a browser that supports getUserMedia (Chrome, Edge, Firefox, Safari all work).

Demos

Demo 1
Hand Tracking
Detect up to two hands in your webcam feed and draw the 21 landmark points per hand along with the full hand skeleton. Learn how MediaPipe Hands normalises landmark coordinates and how to convert them to pixel positions on a canvas.
MediaPipe Hands 21 landmarks Hand skeleton Canvas drawing
Open demo
Demo 2
Face Mesh
Map 468 facial landmarks onto a detected face, with distinct colours highlighting the eyes, eyebrows, nose, lips, and irises. See how FaceMesh's refined landmark model captures fine detail across the entire face surface.
MediaPipe FaceMesh 468 landmarks Region colouring Iris refinement
Open demo
Demo 3
Hands and Face Combined
Run both models simultaneously on the same webcam frame. A side-by-side canvas shows the raw feed on the left and the landmark overlay on the right — hands in green, face in red — so you can compare the raw image with the model output at a glance.
Hands + FaceMesh Dual models Side-by-side layout Async frame pipeline
Open demo
Demo 4
Interaction Basics
Turn landmark coordinates into interactive inputs. Four patterns are shown in a single live view: distance between index fingertip and nose tip, zone detection across horizontal thirds, raised-finger counting, and height-to-colour mapping.
Distance Zone detection Finger count Value mapping
Open demo
Demo 5
Face Instrument
Your face becomes a proximity-triggered control surface. Nine named regions — forehead, eyebrows, eyes, nose, lips, and cheeks — each activate with a distinct colour and ripple effect when the index fingertip comes close.
Proximity triggers Face regions Ripple effects Multi-zone control
Open demo
